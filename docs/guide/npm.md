# NPM

## CMD 命令

### npm 更新

1.npm-check 检查更新

`npm install -g npm-check`
`npm-check`

2.npm-upgrade更新

`npm install -g npm-upgrade`
`npm-upgrade`

3.更新全局包：
`npm update  -g`

4.更新生产环境依赖包：
`npm update  --save`

5.更新开发环境依赖包：
`npm update  --save-dev`

### 安装依赖

npm install 依赖  --save-dev

1. 安装"npm-check-updates"模块
`npm install -g npm-check-updates`

## NCU

ncu  // 检查更新
ncu -u  //执行更新
